 Portfolio


Voici mon **tout premier projet web** au sein du BUT Informatique, réalisé dans le cadre de mon cours de **développement web**.

Il s'agit d'un portfolio personnel mutli-pages.


Ce projet m'a permis :


    D'acquérir des bases solides en HTML, concernant la structure d'un site.

    D'apprendre à appliquer des styles avec CSS et de maitriser les Flex-box.

    D'apprendre les bases du PHP grâce à la mise en place d'un formulaire de contact.

Pour consulter ce portfolio :


Mode simple :

    Téléchargez le dossier du projet et ouvrez index.html dans votre navigateur préféré.

-------------------------------------------------------------------

Mode avancé (pour un envoi du formulaire de contact fonctionnel) : 

Prérequis : vérifiez que PHP est bien installé sur votre machine. (**php** -v dans un terminal ou dans l'invite de commande)


-Sous Windows


    Ouvrez le dossier de votre projet dans l'Explorateur de fichiers.

    Dans la barre d'adresse en haut (là où est écrit le chemin du dossier), tapez cmd et appuyez sur Entrée. Cela ouvre l'invite de commande directement au bon endroit.

    Lancez le serveur avec cette commande : php -S localhost:8000

    Consultez le site : Ouvrez votre navigateur et allez sur l'adresse http://localhost:8000.

------------------------------------------------------------------

-Sous Linux (Ubuntu, Debian, etc.)


    Ouvrez un terminal (souvent Ctrl + Alt + T).

    Déplacez-vous dans le dossier de votre projet : cd /chemin/vers/votre/portfolio (Astuce : vous pouvez taper cd puis glisser-déposer votre dossier directement dans le terminal).

    Lancez le serveur : php -S localhost:8000

    Consultez le site : Allez sur http://localhost:8000 dans votre navigateur.
